# GitHub'a push

Repo sıfırdan bağlandı, ilk commit atıldı. Push'u siz tamamlayın (GitHub girişi gerekebilir).

Terminalde proje klasöründe:

```bash
git branch -M main
git push -u origin main
```

- **HTTPS** kullanıyorsanız: GitHub kullanıcı adı + Personal Access Token (şifre yerine) istenir.
- **SSH** kullanmak için: `git remote set-url origin git@github.com:Ismailjobs/rumpel.git` sonra `git push -u origin main`
